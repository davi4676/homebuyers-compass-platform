# Assets

Add the following files for the Expo app (or copy from a project created with `npx create-expo-app@latest`):

- **icon.png** – 1024×1024 app icon
- **splash-icon.png** – Splash screen image
- **adaptive-icon.png** – 1024×1024 Android adaptive icon foreground
- **favicon.png** – Web favicon

Until these exist, you can temporarily remove or comment out the `icon`, `splash`, and `adaptiveIcon` entries in `app.json` to run the app.

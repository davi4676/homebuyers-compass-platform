Journey-tab editorial images
============================

The customized-journey "Today" tab currently reuses the landing-page asset
`/images/quiet-editorial-interior-v2.jpg` as a soft placeholder. When dedicated
journey imagery is ready, drop the final files at the paths below — the code
already comments these as designer notes at the corresponding render sites.

Recommended assets (all warm, softly lit, editorial — see spec):

  /images/journey/journey-status-01.jpg
      Small portrait/landscape thumbnail rendered inside the journey-status
      card (right column, beside the phase progress ring).
      Suggested subject: calm interior, soft daylight, neutral living room.

  /images/journey/home-planning-01.jpg
      Optional secondary asset for the primary Today module if you decide to
      add a thumbnail there. Suggested subject: first-time buyer planning
      moment — laptop with budgeting notes, paperwork on a table.

  /images/journey/budget-table-01.jpg
      Rendered inside the "Prepare for your next lender conversation" card
      (right-side image strip). Suggested subject: notebook + laptop on a calm
      planning table with warm light.

  /images/journey/calm-interior-01.jpg
      Optional swap-in for any other lower-priority editorial moment.

Style guardrails (per design spec):
  - Avoid generic stock photos of people shaking hands, oversized houses, or
    aggressive real-estate imagery.
  - Prefer calm home interiors, neutral living rooms, soft daylight, paperwork
    on a table, or a first-time buyer planning moment.
  - Rounded corners are applied in CSS — supply rectangular source files.
  - 16:9 or 4:3 source aspect ratio works best for the current containers.
